react开发环境
* Create-react-app
  * facebook官方推荐脚手架，使用他我们不需要在手动配置webpack和eslint等
  * npm start开启调试页面
* 如何安装和使用第三方库
  * npm install redux --save
* 如何定制化配置
  * npm run eject 弹出配置文件，可以自定义webpack（Jest、Babel、ESLint）,发现多了config和script文件夹，同时package.json多了很多的依赖性，之前都是封装在react script内部的，之后我们便可以自定义配置了
